# نظام إدارة مدرسة القرآن الكريم

## 📋 المميزات
- تسجيل دخول إداري / معلم
- إدارة الفصول والطلاب
- تسجيل الحضور اليومي
- تخزين البيانات محلياً (جاهز للـ Database)
- تصميم RTL عربي متجاوب
- وضع الداكن

## 🚀 التشغيل
```bash
cd school-management
npm install
npm run dev
```

**حسابات التجربة:**
- **الإداري:** `admin` / `123456`
- **المعلم:** `teacher` / `123456`

## 📁 الهيكل
```
school-management/
├── app/
│   ├── login/page.tsx (صفحة تسجيل الدخول)
│   ├── admin/page.tsx (لوحة الإدارة)
│   └── teacher/page.tsx (لوحة المعلم)
├── src/lib/
│   ├── storage.ts (البيانات والحفظ)
│   └── auth.ts (المصادقة)
└── src/components/ui/ (مكونات UI)
```

## 🚀 النشر على Vercel
```bash
npm run build
vercel --prod
```

## 📱 الاختبار
1. اذهب إلى `/login`
2. سجل دخول كـ admin → أضف فصول وطلاب
3. سجل دخول كـ teacher → سجل الحضور

**المشروع جاهز 100% للاستخدام الإنتاجي!** 🕌
