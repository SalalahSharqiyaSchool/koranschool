# نظام رصد الغياب — مدرسة القرآن

وزارة التعليم | محافظة ظفار | العام الدراسي 2025-2026

## المميزات
- تسجيل الغياب اليومي لكل مجموعة وشعبة
- إدارة الطلاب (إضافة / تعديل / حذف / استيراد جماعي)
- تقارير مفصلة (طالب أو صف كامل) مع إمكانية الطباعة
- تعديل وحذف سجلات الغياب
- حسابات منفصلة لكل معلم بكلمة مرور مشفرة
- حفظ البيانات مباشرة على GitHub
- يعمل بالتزامن — كل معلم يعدّل ملف مجموعته المستقل

## الإعداد الأولي

### 1. رفع المشروع على GitHub
```bash
git init
git remote add origin https://github.com/SalalahSharqiyaSchool/studentabsent.git
git add .
git commit -m "المشروع الأولي"
git push -u origin main
```

### 2. إعداد بيانات المعلمين
- افتح الموقع وانتقل إلى `/setup`
- أدخل GitHub Token واضغط "بدء الإعداد"
- سيُنشئ ملف `data/teachers.json` تلقائياً

### 3. تسجيل الدخول
| المستخدم | الاسم | كلمة المرور |
|---|---|---|
| faisal | فيصل العريبي | admin |
| teacher1 | معلم 1 | 1234 |
| teacher2 | معلم 2 | 12345 |

### 4. النشر على Vercel
- اربط المستودع بـ Vercel
- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

## هيكلة البيانات على GitHub

```
data/
  teachers.json              ← حسابات المعلمين (مشفرة)
  students/
    مجموعة-شعبة.json         ← قائمة طلاب كل مجموعة
  attendance/
    مجموعة-شعبة.json         ← سجل غياب كل مجموعة
```

## ملاحظة أمنية
- GitHub Token يُحفظ محلياً في المتصفح فقط
- كلمات المرور مشفرة بـ SHA-256
- لا يُرسل أي بيانات لخوادم خارجية
